Piano MIDI Viewer

A tool for music teachers, students, and content creators.

Free, open source, no subscription, no catch.

Get started

  1. Download PianoMIDIViewer.exe
  2. Run the file
  3. Windows SmartScreen may pop up — click "More info", then "Run anyway"
  4. That's it! The app is ready to use.

SmartScreen warns about unsigned apps. Piano MIDI Viewer is open source and safe — you can read every line of code.

  1. Download PianoMIDIViewer.dmg
  2. Open the DMG and drag Piano MIDI Viewer to your Applications folder
  3. Open Terminal (search for it in Spotlight) and paste this command: xattr -cr /Applications/PianoMIDIViewer.app
  4. Now open the app from Applications — it's ready!

The xattr command removes macOS quarantine flags. Apple requires a $99/year fee to sign apps — we'd rather keep the app free.

  1. Download PianoMIDIViewer.AppImage
  2. Right-click the file → Properties → mark it as executable
  3. Double-click to run!

Or in terminal: chmod +x PianoMIDIViewer.AppImage && ./PianoMIDIViewer.AppImage
